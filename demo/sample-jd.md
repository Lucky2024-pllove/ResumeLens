# 岗位要求（演示）

- **岗位名称**：后端开发工程师（业务中台）
- **级别**：高级（P6 相当）
- **地点**：上海（可接受短期远程，需能到岗）
- **学历**：本科及以上，计算机或相关专业优先；**学历权重**：20%
- **年限**：5 年及以上后端经验；**经验权重**：35%
- **核心技能**（**能力权重**：40%）：
  - Go：熟练，需有线上高并发服务经验
  - 分布式系统：缓存、消息队列、分库分表至少各一项落地经验
  - Kubernetes：能独立排查部署与网络问题
- **加分**：Rust、可观测性（Prometheus/Grafana）、成本优化经验
- **软技能**（**软技能权重**：5%）：跨团队沟通、需求澄清与文档习惯
- **硬性门槛**：无 Go 生产经验不进入终面；最近一段工作不少于 18 个月
- **评估权重汇总**：学历 20% / 能力 40% / 经验 35% / 软技能 5%（与 SKILL 默认示例略有不同，演示「可配置权重」）
